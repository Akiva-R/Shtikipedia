## Proposal

The Chazon ish says that ברי ושמא create a חזקה that it is the ברי's.
## Source

1. The Gemara says a woman can be הוציא כתובה with a ברי ושמא.
2. The אחרונים ask, this is a ברי גרוע and a שמא טוב, how can we say this is a proof to take from a מוחזק?
3. Most אחרונים answer along the lines that if he is a שמא, then there is no נטען, so there is no מוחזק.
4. The חזון איש explains that חזקה is פשטא דמילתא, and if ברי ושמא then the פשטות is that is him, so he is מוחזק and the other is the מוציא.
