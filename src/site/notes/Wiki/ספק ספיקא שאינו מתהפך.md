## Proposal

The ש"ך explains that a [[Wiki/ספק ספיקא\|ספק ספיקא]] needs to be מתהפך, because מאי חזית.
## Source

1. Many ראשונים say that if a ספק ספיקה can only be asked from one direction, it is not considered a ספק ספיקה.
2. The שך (יו"ד קי כללי ספק ספיקה יג) explains that the reason is because מאי חזית to start form this direction, start from the other direction.
3. Based on this he says that if their is a reason you can only ask from one direction, it would be considered a ספק ספיקה. He brings to proofs from the גמרא to this:
	1. On ב"ק דף יא:א we talk about טומאה if there was a ולד, and if יצא רובו, and we consider it a ס"ס even though it is not מתהפך, because we have to start by asking if their is a ולד at all.
	2. On גיטין דף מה:ב we suggestg you can use a ספר תורה bought from an עכו"ם becuases there is a ספק it was written by a jew, and even if it was written by an עכו"ם, maybe it was written to sell to a jew. This is not מתהפף, but it only makes sence to start by asking who wrote it.
