## Proposal

Although we  פסקן like שמואל that אין הולכין בממון אחר הרוב, some hold that that is only by a רובא דליתא לקמן, but by a רובא דאיתא לקמן, we can be מוציא with a רוב.
## Source

1. The גמרא in חולין יא:א says we learn רוב from אחרי רבים להטות. 
2. The גמרא asks that only helps for a רובא דאיתא לקמן, but what about a רובא דליתא לקמן? It does not bring a source.
3. Rashi (שם יב:א) says that either רובא דליתא לקמן is learned from איתא לקמן, or it is a הלכה למשה מסיני.
4. If we hold that we can't be הוציא ממון with רוב, then how can we learn it from אחרי רבים להטות, of course רוב דיינים can be הוציא ממון.
5. The תרוה"ד (שי"ד), and the שטמ"ק (ב"ב צב) bring from the גליון תוס and the רשב"ם that we can be הוציא with a רובא דאיתא לקמן, like רוב דיינים.
## Alternative Explanations

+ Tos ב"ק כז explains that by רוב דיינים, it is like the מיועט does not exist.
